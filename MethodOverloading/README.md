# Method Overloading
A console-based method overloading demonstration built in C#.

## Features
- Four overloaded Add methods with different parameter types
- Demonstrates C# automatic method selection based on arguments
- Covers int, double, and string overloads

## Example
```
Add(3, 5) = 8
Add(1.5, 2.3) = 3.8
Add(1, 2, 3) = 6
Add("Hello", " World") = Hello World
```
