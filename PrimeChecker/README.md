# Prime Checker
A console-based prime number checker built in C#.

## Features
- User input with validation
- Edge case handling for numbers smaller than 2
- Prime checking loop with early exit via break
- Clear output messages

## Example
```
Enter a number to check if it's a prime number: 7
7 is a prime number!

Enter a number to check if it's a prime number: 9
9 is not a prime number!
```
