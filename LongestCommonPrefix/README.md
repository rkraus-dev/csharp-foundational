# Longest Common Prefix
A console-based longest common prefix finder built in C#.

## Features
- Finds the longest common prefix across all strings in an array
- Returns empty string if no common prefix exists
- Stops immediately when a mismatch is found

## Example
``` 
Input: ["flower", "flow", "flight"]
Longest common prefix: "fl"

Input: ["dog", "racecar", "car"]
Longest common prefix: ""
```
