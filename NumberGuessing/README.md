# Number Guessing Game
A console-based number guessing game built in C#.

## Features
- Randomly generated target number between 1 and 100
- Input validation with re-prompt on invalid entry
- Hints for too high / too low guesses
- Attempt counter displayed at the end

## Example
```
Guess a number between 1 and 100: 50
Too High! Try again!
Guess a number between 1 and 100: 25
Too Low! Try again
Guess a number between 1 and 100: 37
Correct! You guessed it in 3 attempt(s)!
```
