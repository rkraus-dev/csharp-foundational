# Find Duplicates
A console-based duplicate finder built in C#.

## Features
- Finds all duplicate values in an array
- Each duplicate is only reported once
- Uses HashSet for efficient lookup

## Example
```
Input: [1, 2, 3, 2, 4, 3, 5]
Duplicates: 2, 3
```

03 - Arrays and Collections

## Topics Practiced
- HashSet<T> for duplicate detection
- foreach loop for array iteration
- Add() return value (true/false) for duplicate check
- string.Join for collection output
