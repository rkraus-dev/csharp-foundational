# Remove Duplicates
A console-based duplicate remover built in C#.

## Features
- Removes duplicate values from an array
- Preserves the order of first occurrences
- Uses HashSet for efficient duplicate detection

## Example
```
Input: [1, 2, 3, 2, 4, 3, 5]
Output: [1, 2, 3, 4, 5]
```

## Topics Practiced
- HashSet<int> for duplicate detection via Add() return value
- List<int> for maintaining insertion order
- foreach loop for array iteration
- string.Join for collection output
