# C# Foundational ![Status](https://img.shields.io/badge/in%20progress-yellow)

My solutions and exercises from the **FCC Foundational C# with Microsoft** 
certification path. Every folder is a standalone exercise or project, 
organized by chapter.

> **Note:** These exercises are not part of the official course material.
> They were designed by me — with the support of AI tools (Claude & Gemini) 
> — to reinforce and deepen each chapter's concepts, combining classic coding 
> challenges (e.g. FizzBuzz, Two Sum, Binary Search) with custom projects 
> that cover the same topics. The goal was deliberate practice beyond the 
> curriculum, not reproduction of course content.
