# Missing Number
A console-based missing number finder built in C#.

## Features
- Finds the missing number in an array of n distinct numbers
- Uses the expected sum formula: n * (n + 1) / 2
- Displays input array and missing number

## Example
```
Input:  3, 0, 1
Missing number: 2
```
