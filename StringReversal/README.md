# String Reversal
A console-based string reversal exercise built in C#.

## Features
- Reverses a user-provided string without built-in Reverse methods
- Trims whitespace from input
- Iterates string backwards using index

## Example
```
Enter a string: Hello
Reversed: olleH
```

## Chapter
04 - Strings and Methods

## Topics Practiced
- for loop with reverse iteration
- String indexing with userInput[i]
- Null-coalescing operator ??
- Trim() for whitespace removal
