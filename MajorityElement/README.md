# Majority Element
A console-based majority element finder built in C#.

## Features
- Finds the element that appears more than n/2 times
- Uses a Dictionary to count occurrences
- Displays the majority element

## Example
```
[3, 2, 3]
Majority Element: 3
```

## Chapter
03 - Arrays and Collections

## Topics Practiced
- Dictionary<int, int> for counting occurrences
- foreach over array to populate dictionary
- foreach over KeyValuePair to find majority
- ContainsKey() for dictionary lookup
- n/2 threshold comparison
