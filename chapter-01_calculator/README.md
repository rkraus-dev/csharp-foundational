# Calculator

A console-based calculator built in C#.

## Features

- Addition, Subtraction, Multiplication, Division, Modulo
- Input validation with error handling
- Division by zero protection
- Formatted output

## Example

```
Enter first number: 10
Enter second number: 3
Enter operator (+, -, *, /, %): *
10 * 3 = 30
```
