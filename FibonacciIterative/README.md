# Fibonacci Iterative
A console-based iterative Fibonacci sequence generator built in C#.

## Features
- User defines how many Fibonacci numbers to generate
- Calculates sequence iteratively using two variables
- Comma-separated output without trailing comma

## Example
```
How many Fibonacci numbers?: 8
Fibonacci: 0, 1, 1, 2, 3, 5, 8, 13
```
