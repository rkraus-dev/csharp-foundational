# 2D Grid
A console-based 2D array exercise built in C#.

## Features
- Creates and displays a 3x3 grid
- Calculates the sum of all values
- Finds the maximum value in the grid

## Example
```
Grid:
1 2 3
4 5 6
7 8 9
Sum: 45
Max: 9
```

## Chapter
03 - Arrays and Collections

## Topics Practiced
- 2D array declaration and initialization
- Nested for loops for grid traversal
- Sum accumulation with +=
- Max value tracking with if comparison
