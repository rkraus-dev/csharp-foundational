# First Unique Character
A console-based first unique character finder built in C#.

## Features
- Finds the first character that appears only once in a string
- Uses a Dictionary to count character occurrences
- Handles case where no unique character exists

## Example
```
Enter a string: leetcode
First unique character: l

Enter a string: aabb
No unique character found!
```

04 - Strings and Methods

## Topics Practiced
- Dictionary<char, int> for character counting
- foreach loop to populate dictionary
- for loop with break to find first unique character
- bool flag to track if unique character was found
- ContainsKey() for dictionary lookup
