# Move Zeroes
A console-based array manipulation exercise built in C#.

## Features
- Moves all zeroes to the end of an array
- Preserves the order of non-zero elements
- Displays input and output array

## Example
```
Input:  0, 1, 0, 3, 12
Output: 1, 3, 12, 0, 0
```  

## Chapter
03 - Arrays and Collections

## Topics Practiced
- int array declaration
- for loop with index
- Two-pointer technique (insertPos)
- string.Join for array output
