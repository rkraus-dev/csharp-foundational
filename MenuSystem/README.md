# Menu System
A console-based interactive menu system built in C#.

## Features
- Interactive menu with 3 options
- Input validation for non-numeric input
- Loops until user selects Exit
- Each option handled by its own method

## Example
```
=== Main Menu ===
[1] Say Hello
[2] Add Two Numbers
[3] Check Even or Odd
[0] Exit
Choose an option: 2
Please input first Number: 5
Please input second Number: 3
Result: 8
=== Main Menu ===
Choose an option: 0
Goodbye!`
```

## Topics Practiced
- do while loop for menu navigation
- switch statement for option handling
- void methods for each menu action
- TryParse with bool validation
- Modulo operator % for even/odd check
