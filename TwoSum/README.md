# Two Sum
A console-based two sum solver built in C#.

## Features
- Finds two indices whose values add up to a target
- Uses nested loops for pair comparison
- Displays matching indices and values

## Example
```
[2, 7, 11, 15], target = 9
Indices: [0, 1] (2 + 7 = 9)
```

## Topics Practiced
- Nested for loops for pair comparison
- Inner loop starting at i+1 to avoid duplicates
- string.Join for array output
