# Bubble Sort
A console-based bubble sort implementation built in C#.

## Features
- Sorts an integer array using the Bubble Sort algorithm
- Compares adjacent elements and swaps if out of order
- Displays input and sorted output

## Example
```
Input: [5, 3, 8, 1, 2]
Output: [1, 2, 3, 5, 8]
```

## Chapter
03 - Arrays and Collections

## Topics Practiced
- Nested for loops for sorting passes
- Adjacent element comparison with index j and j+1
- Swap using a temporary variable
- string.Join for array output
