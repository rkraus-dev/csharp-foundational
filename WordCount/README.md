# Word Count
A console-based word frequency counter built in C#.

## Features
- Splits user input into individual words
- Counts occurrences of each word case-insensitively
- Handles multiple spaces between words
- Displays each word with its count

## Example
```
Enter a text: the quick brown fox jumps over the lazy dog the
the: 3
quick: 1
brown: 1
fox: 1
jumps: 1
over: 1
lazy: 1
dog: 1
```
